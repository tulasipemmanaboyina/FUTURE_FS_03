------------------------Brew Haven Café Website-----------------------------
**Project Overview
Brew Haven Café is a modern and responsive local café business website developed as part of the Future Interns Full Stack Web Development Internship – Task 3.
This website was created to help a café business improve its online presence, attract customers, showcase menu items, and provide an easy way for customers to connect and place orders.
The project focuses on real-world business website design with a professional user interface and customer-friendly features.

**Features

Responsive modern website design
Professional navigation bar
Hero section with café branding
About café section
Coffee menu cards
WhatsApp ordering feature
Special offers section
Customer reviews section
Image gallery
Working contact form using Formspree
Embedded Google Maps location
Mobile-friendly responsive layout

**Technologies Used
HTML5
CSS3
JavaScript
Font Awesome
Formspree
Google Maps Embed

**Website Purpose
This website was built to help a local café business:
Build a professional online presence
Showcase coffee menu and services
Improve customer trust and engagement
Allow direct WhatsApp ordering
Help customers find the café location easily
Increase customer interaction through contact forms
